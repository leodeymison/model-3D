=> @types:
    São extensões de pacotres npm já existentes para 
    que o typescript suporte propriedades daquele pacotres

    exemplo: @types/node = Suporta algumas propriedades do node
    como o "require".


=> webpack:
    É um empacotador de módulos para aplicações js
    que ler e cria novos pacotes para serem usados.


=> webpack-cli:
    Ferramenta que nos da acesso ao comandos ao complite
    do webpack


=> webpack-dev-server:
    Um pacote que oferece um servidor, que cria um único 
    arquivo com todos os camandos (bulder.js)


=> webpack-merge:
    Mesclagem de pacotes web
